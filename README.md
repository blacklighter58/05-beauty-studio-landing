# Lumi Beauty Landing

Lumi Beauty Landing is a responsive landing page for a small beauty studio focused on natural-looking brows, lashes, makeup and light care services. The page presents services, prices, booking flow and FAQ in a calm premium style without using real photos.

## Project Goal

The goal of this project is to create a realistic commercial landing page for a beauty studio where visitors can quickly understand the services, compare prices and move toward booking. The content focuses on an аккуратный everyday look, calm communication and clear service details.

## Features

- Hero section with a specific offer for an аккуратный daily look
- Service cards with practical descriptions for brows, lashes, makeup and packages
- Studio details about consultation, shade choice and aftercare
- Decorative portfolio placeholders with realistic work categories
- Step-by-step booking explanation
- Interactive price selection before booking
- Human-sounding demo reviews without real names
- FAQ accordion with concrete questions
- Responsive layout for desktop, tablet and mobile
- Gentle hover effects, section reveal animation and calm decorative details
