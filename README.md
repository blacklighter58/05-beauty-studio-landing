# Lumi Beauty Landing

Lumi Beauty Landing is a modern responsive landing page for a premium beauty studio. The page presents brow, lash, makeup and care services in a soft, elegant visual style.

## Project Goal

The goal of this project is to practice building a complete commercial landing page with clear structure, aesthetic presentation, responsive layout and small JavaScript interactions that help visitors book a service online.

## Features

- Light premium beauty studio design
- Decorative soft background details
- Responsive header with mobile menu
- Hero section with clear offer and call-to-action buttons
- Compact studio notes and trust details
- Service cards for brows, lashes, makeup and complex care
- Benefits section with studio values
- Decorative portfolio placeholder gallery
- Step-by-step booking process
- Pricing section with clear starting prices
- Interactive service price selection before booking
- Demo review cards
- Clickable FAQ accordion
- Contact CTA section
- Smooth section reveal animation on scroll
- Interactive hover and click states for buttons and cards
